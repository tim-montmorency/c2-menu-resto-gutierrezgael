## Description
Mon menu est très minimal, mais raffiné. Il contient des confiseries magnifiques, qui amèneront du plaisir à tous mes clients.
Les 3 catégories sont:
Les Crémeux (Tarte à la crème, crème glacée à la vanille française, crème fouettée).
Les Sucrosités (Sucette gourmande, brochette de bonbecs, dragées).
Les Fluides (Soda à la framboise, smoothie à la banane, lait aux fraises).

Tarte à la crème : Une tarte très crémeuse! 18$
Crème glacée à la vanille française : Une saveur de vanille rafraîchissante! 12$
Crème fouettée : Une crème fouettée, accompagnée de petits fruits exquis, qui vous sucre le bec! 8$

Sucette gourmande : Une sucette de couleur arc-en-ciel qui vous régalera! 12$
Brochette de bonbecs : Une brochette extravagante de jujubes multicolores!  6$
Dragées : Aussi connues sous le nom de 'Jelly Beans', ces dragées vous enverront au paradis, garanti! 15$

Soda à la framboise : Une liqueur amusante faite à base de framboises cueillies à Marseille! 6$
Smoothie à la banane : Un breuvage tropical, qui vous fera penser aux plages de la Martinique! 8$
Lait aux fraises: Une boisson laitière aux fraises fraîches! 10$
